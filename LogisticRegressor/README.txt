Instructions to run logreg.py

1) To run the program to generate the weights.csv for the sample data

python logreg.py [sample data file]

eg:- python logreg.py ldata.csv
     python logreg.py nldata.csv
	 
2) To run the program to generate visualization of the linear classification 

python logreg.py [sample data file] [weights vector file]

eg:- python logreg.py ldata.csv weights.csv
     python logreg.py nldata.csv weights.csv
