This a program to find the solution for a dice which is positioned with 1 
facing up to reach the solution to the destination by moving in the maze
along by changing its allignments of numbers through the course of action
to find the solution in the maze. It cant move to the position of a obstacle 
and there by it has to find the solution in an optimal way.

There are five mazes provides as Puzzlex.txt where x lies from [1-5]

To run the program enter
python sdmaze.py [FileName]
